3)
	Тег является основным элементом разметки.
	Допускается вкладывать в парный тег (контейнер) другие теги.
	Атрибуты позволяют изменять свойства и поведение элемента.
	Текст отображается на сайте.
*) Атрибут
	Атрибут является частью тега, позволяющей менять его поведение или добавлять метаданные. Атрибут всегда представлен в виде название=значение, которые определяют соответственно идентификатор атрибута и присвоенное ему значение.
4) Типы тегов
	Теги в зависимости от названия имеют разный функционал.

5) Теги верхнего уровня
	Тег <html> является корневым элементом документа.
	Тег <head>  содержит техническую информацию о странице: заголовок, описание, ключевые слова для поисковых машин, кодировку и не отображается в окне браузера
	Тег <body> предназначен для хранения содержания веб-страницы

6) Теги заголовка документа
	<head> - Элемент-контейнер для метаданных HTML-документа, таких как<title>, <meta>, <link>.
	<meta> - Используется для хранения дополнительной информации о странице. Эту информацию используют браузеры для обработки страницы, а поисковые системы — для ее индексации, может быть несколько тегов <meta>, так как в зависимости от используемых атрибутов они несут разную информацию.
	<link> - Определяет отношения между документом и внешним ресурсом. Также используется для подключения внешних файлов.
	href - Путь к связываемому файлу.

*) ссылки
	Чтобы указать ссылку, нужно использовать тег <a> и атрибут href значения которого - путь до сайта или файла. Если указать id блока то страница проскролится к нему. Чтобы ссылка открывалась в другом окне, нужно добавить атрибут target со значениям _blank для ссылки.
	target Указывает на то, в каком окне должен открываться документ, к которому ведет ссылка.
